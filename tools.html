GNU Toolchain for ARC

The ARC GNU Toolchain offers all of the benefits of open source tools, including complete source code and a large install base. The tools include the GCC compiler and GDB debugger as well a number of utilities and libraries that make up a complete software toolchain. Because source code for the GNU t oolchain is always available, the ARC GNU Toolchain is configurable, allowing tailoring, customization, and expansion of the tools, libraries, debugger, and board support to meet user-specific requirements. The GNU command line tools for ARC Processors and the GNU Toolchain for ARC Processors with Eclipse IDE can be downloaded from GitHub.
