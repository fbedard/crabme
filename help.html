embARC FAQ - Frequently Asked Questions

What is embARC?

embARC is the community of developers working to add support for the ARC architecture to various embedded open source projects. It consists of Synopsys Open Source engineers, Synopsys Ecosystem Partners, ARC licensees, their customers and anyone else interested in porting their software to ARC.

embARC Open Software Platform (OSP) is an open software platform maintained by Synopsys Open Source Developers that provides a comprehensive software stack for embedded applications, based on industry-leading open source solutions.

embARC OSP leverages the versatility of the DesignWare® ARC® EM Starter Kit to enable rapid software development, code porting, software debugging, and profiling of applications on a n FPGA prototype. The platform is extensible using a wide range of Digilent Pmod™ modules (sensor, WiFi, RF, more) to create a broad range of IoT applications.

What development tools should I use with embARC?

Synopsys's premium toolchain for use with the ARC EM family of processors is the ARC MetaWare Development Toolkit. Licensing information is available at https://www.synopsys.com/dw/ipdir.php?ds=sw_metaware. Synopsys's free GNU toolchain may also be used for embARC development, but some options and extensions of the configurable ARC EM family of processors may only be supported by the MetaWare Development Toolkit.
